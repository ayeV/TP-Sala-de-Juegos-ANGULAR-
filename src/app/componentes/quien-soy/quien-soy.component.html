Juego propio: Ahorcado

Se le presenta al usuario un teclado compuesto por botones, al clickear uno, si la letra esta contenida en la palabra secreta, se adiciona la misma, si la letra no esta contenida entonces se dibuja una parte del ahorcado y asi sucesivamente. Cuando el dibujo del ahorcado esta completo el usuario perdió el juego y no puede volver a ingresar una letra. Si el usuario logró ganar entonces se le suman 10 puntos en la base de datos.
